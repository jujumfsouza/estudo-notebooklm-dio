# estudos-naughtydogs-notebooklm

# ⭐Contexto e Objetivos:
Estudos dos principais pilares e sua metodologia de criação de jogos da Naughty Dogs. Desafio de projeto da DIO "Dados e GenAI", ultilizando a ferramenta nootbooklm.

Objetivo em adquirir conhecimento das técnicas e conceitos apresentados durante esta jornada, sendo capaz de aprimorar e desenvolver habilidades para criação e design de jogos.
